sf3-ecommerce
=============

A next generation of ecommerce build with symfony 3 and more.


* PHP 7.1
* NGINX
* MySQL
* Redis
* ELK (coming soon)
* REST API (coming soon)
* React JS (coming soon)

Start Env
---------

1) Install Docker
2) docker-composer build
3) docker-composer up

Stop Env
--------

1) docker-compose stop


Build project
-------------

1) Install Composer
2) Composer install


Access to
---------

* Web : localhost
* phpMyAdmin : localhost:8080
